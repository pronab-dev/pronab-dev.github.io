## 🚀 Run Locally

No installation or build tools are required.

### Option 1

Simply open:

```
index.html
```

in your browser.

### Option 2 (Recommended)

```bash
git clone https://github.com/pronab-dev/pronab-dev.github.io.git

cd pronab-dev.github.io

python -m http.server 8000
```

Then open:

```
http://localhost:8000
```
